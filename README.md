## Installation

RainbowLabel is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'RainbowLabel', git => 'https://github.com/QiushiZhao/RainbowLabel.git'
```
